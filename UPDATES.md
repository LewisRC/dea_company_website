## 技术特点

### 响应式设计
- 桌面端：完整的下拉菜单和所有功能
- 移动端：折叠式菜单，支持子菜单展开

### 用户体验优化
- 鼠标悬停即可显示下拉菜单（桌面端）
- 菜单项高亮显示当前页面
- 平滑的过渡动画
- 清晰的视觉层次

### 性能优化
- 使用 Next.js Image 组件自动优化图片
- 代码分割和懒加载
- CSS 模块化和工具类

## 本地运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 访问 http://localhost:3000
```

- **React**: 19.2.0
- **样式**: Tailwind CSS 4.1.9
- **图标**: Lucide React
- **TypeScript**: 完整的类型支持
